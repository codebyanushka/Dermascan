# DermaCam 🔬
### AI-Powered Dermatologist — Hackathon Project

© 2026 Anushka Tiwari. All Rights Reserved.

## What it does
- 🔬 Detects 31+ skin conditions using DinoV2 (96.48% accuracy)
- 💊 Personalized treatment plan + skincare routine
- 🤳 Live camera scan — no upload needed
- 📋 Skin history tracker
- 📍 Find nearby dermatologists

## Tech Stack
DinoV2 · Llama 4 Vision · Groq · Flask · WebRTC · HuggingFace
